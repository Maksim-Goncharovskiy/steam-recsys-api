# steam-recsys-api
API-сервис для предоставления content-based рекомендаций игр из Steam.

## Документация API
API предоставляет 2 POST-эндпойнта для получения рекомендаций с идентичным интерфейсом:
```
POST /recsys/v1/
    Описание:
        - первая версия рекомендательной системе на базе TF-IDF
    Параметры:
        - liked_games_ids (list[int]): id понравившихся пользователю игр
        - m (int): количество игр, которое должно быть в рекомендации
    Возвращает:
        - json со структурой:
        [
            {
                "rec": id рекомендуемой игры,
                "src": id понравившейся пользотелю игры, на которую похожа рекомендованная
            },
            ...
        ]


POST /recsys/v2/
    Описание:
        - вторая версия рекомендательной системы на базе Doc2Vec и с обработкой категориальных признаков
    Параметры:
        - liked_games_ids (list[int]): id понравившихся пользователю игр
        - m (int): количество игр, которое должно быть в рекомендации
    Возвращает:
        - json со структурой:
        [
            {
                "rec": id рекомендуемой игры,
                "src": id понравившейся пользотелю игры, на которую похожа рекомендованная
            },
            ...
        ]
```

Пример входных данных:
```json
{
  "liked_games_ids": [
    20200
  ],
  "m": 15
}
```

Пример выполняемого запроса к API:
```bash
$curl -X 'POST' \
  'http://localhost:8000/recsys/v1/' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -d '{
  "liked_games_ids": [
    20200
  ],
  "m": 15
}'
```


Пример ответа:
```json
[
  {
    "rec": 43683,
    "src": 20200
  },
  {
    "rec": 795,
    "src": 20200
  },
  {
    "rec": 71681,
    "src": 20200
  },
  {
    "rec": 44619,
    "src": 20200
  },
  ...
]
```
